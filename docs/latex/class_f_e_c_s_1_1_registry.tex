\doxysubsection{Detailed Description}
Acts as a main interface for the ECS, manages a {\ttfamily \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Local\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}} environment for entities, and component manipulation. 

The registry handes entity creation and destruction, component attachment/detachment, exposes access to Component Pool and Views
