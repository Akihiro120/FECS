\doxysubsection{Detailed Description}
Provides static access to component pools and handles entity cleanup. 

The \doxylink{class_f_e_c_s_1_1_manager_1_1_component_manager}{Component\+Manager} is a singleton-\/like utility with static methods and per-\/type static storage (via Templates) to manage components, and versions efficiently.
