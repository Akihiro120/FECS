\doxylink{class_f_e_c_s_1_1_container_1_1_sparse_set}{Sparse\+Set} stores components in a densely packed array while allowing fast indexed lookup via sparse indirection. 

Uses paged sparse arrays and dense vectors to provide fast O(1) insertion, removal, and access with minimal memory waste.
