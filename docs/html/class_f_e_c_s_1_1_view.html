Detailed Description

template<typename... Components>
class FECS::View< Components >

A view for iterating over entities that have all specified component types.

The View caches matching entities and automatically rebuilds the cache when component versioning changes (i.e, when entities/components are added or removed)

Template Parameters
ComponentsThe component types to filter and access.
